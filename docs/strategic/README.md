# Strategic Documents Overview

**Last Updated:** January 2025

This directory contains strategic planning documents that guide product direction, competitive positioning, and future enhancements. These documents complement the current product state documentation (see [Product Owner Review](../PRODUCT_OWNER_REVIEW.md)).

---

## Document Purposes

### 1. PRODUCT_STRATEGY_MORTGAGE_HEALTH_MONITOR.md

**Purpose:** Historical roadmap that documented the strategic pivot from a one-time calculator to an ongoing mortgage health monitoring platform.

**When to Use:**

- Understand the product's evolution and strategic vision
- Reference the original planning phases (now largely complete)
- Review Phase 3+ enhancement opportunities
- Understand the rationale behind monitoring features

**Current Status:** ✅ Phase 1-2 Complete - Historical planning document showing successful transformation to monitoring platform

**Key Content:**

- Original strategic framework (calculator → platform)
- Phase 1-3 feature recommendations (most implemented)
- Subscription value drivers
- Implementation roadmap (historical)

---

### 2. SUBSCRIPTION_VALUE_FRAMEWORK.md

**Purpose:** Timeless strategic framework for evaluating what creates recurring subscription value vs. one-time calculator value.

**When to Use:**

- Evaluating new features for subscription potential
- Prioritizing features based on recurring value
- Understanding subscription business model principles
- Designing features that drive user retention
- Product strategy discussions

**Current Status:** ✅ Timeless Framework - Still relevant for feature evaluation

**Key Content:**

- Framework for identifying subscription value drivers
- Value scoring system (Data Changes, Proactive, Continuous, etc.)
- Examples of features that create recurring engagement
- Calculator vs. Platform characteristics

---

### 3. AI_INTEGRATION_STRATEGY.md

**Purpose:** Strategic analysis and roadmap for integrating AI/GenAI features into the mortgage application.

**When to Use:**

- Planning AI feature development
- Evaluating AI opportunities and costs
- Understanding AI implementation priorities
- Cost/benefit analysis for AI features
- Safety and compliance considerations for AI

**Current Status:** ✅ Future Planning - No AI features implemented yet

**Key Content:**

- 7 AI integration opportunities with analysis
- Prioritized implementation phases
- Cost analysis and ROI considerations
- Technical architecture recommendations
- Safety, compliance, and risk mitigation

---

### 4. COMPETITIVE_ANALYSIS.md

**Purpose:** Competitive intelligence analysis comparing the product to indirect competitors (ProjectionLab, LunchMoney).

**When to Use:**

- Understanding competitive landscape
- Positioning discussions
- Feature differentiation strategy
- Market analysis and positioning
- Competitive benchmarking

**Current Status:** ⚠️ Requires Periodic Review - Competitive landscape may evolve

**Key Content:**

- Detailed analysis of ProjectionLab (financial planning tool)
- Detailed analysis of LunchMoney (budgeting tool)
- Competitive positioning recommendations
- Market differentiation strategy
- Unique value propositions
